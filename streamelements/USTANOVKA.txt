STARTING SOON — TENNIEL (StreamElements)
=========================================

Экран «Скоро начнём» в стиле Джона Тенниела для оверлея StreamElements.
Анимация кота, Алисы и цитаты — как в OBS-версии.

Сборка пакета:

  cd twitch-starting-soon-tenniel/streamelements
  python3 build_streamelements.py

Готовые файлы — в папке out/


1. СОЗДАТЬ ОВЕРЛЕЙ
------------------
• https://streamelements.com → Overlays → NEW OVERLAY
• Разрешение: 1920 × 1080 → START


2. ДОБАВИТЬ CUSTOM WIDGET
-------------------------
• «+» → STATIC / CUSTOM → Custom Widget
• Растяни виджет на весь холст (1920×1080)
• ПКМ → Edit widget → Code Editor


3. ВСТАВИТЬ КОД
---------------
  HTML   ← out/widget.html
  CSS    ← out/widget.css
  JS     ← out/widget.js
  FIELDS ← out/widget.json

Сохрани (Save).


4. ЗАГРУЗИТЬ СЛОИ И НАСТРОИТЬ
-----------------------------
В левой панели (OPEN EDITOR):

Слои — загрузи из out/assets/:
  • Фон (лес)      → background.jpg
  • Кот — тело     → cat-body-cutout.png
  • Кот — голова   → cat-head-cutout.png
  • Алиса          → alice-cutout.png

Канал:
  • Название канала — текст
  • Аватар канала   → channel-avatar.png (или свой)

Текст:
  • Подпись под заголовком — по желанию
  • Цитата кота — по умолчанию уже заполнена

Сохрани → Preview.


5. ПОДКЛЮЧИТЬ К OBS
-------------------
• Copy overlay URL в редакторе SE
• OBS → Browser Source, 1920×1080
• Подогнать по размеру экрана


ЕСЛИ ЧТО-ТО НЕ ТАК
------------------
• Пустой экран → загрузи все 4 слоя в группе «Слои»
• Нет шрифта → нужен интернет (Google Fonts EB Garamond)
• Блок канала не нужен → очисти название и не загружай аватар
